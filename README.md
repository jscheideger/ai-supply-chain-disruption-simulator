# AI-Based Supply Chain Disruption Simulator

This project integrates discrete-event simulation with supervised machine learning to model and predict supply chain disruption impacts.

## Features
- Multi-node supply chain simulation
- Disruption injection engine
- Stockout classification model
- Cost impact regression model

## Tech Stack
- Python
- NetworkX
- scikit-learn
- Pandas

## How to Run
1. Create virtual environment
2. pip install -r requirements.txt
3. python run_demo.py
4. python run_experiments.py
